javaSE基础-方法、运算符

java 运算符 方法
学习笔记
发布日期:   2020-04-19

一、方法

​ 方法是完成特定功能的代码块。

​ 很多语言中都有方法，方法也叫作函数， 只不过在java中一般叫做方法。

1、方法的定义格式：

​ 修饰符 返回值类型 方法名(参数列表) {

​ 方法体;

​ return 返回值;

​ }

​ 格式解释：

​ 修饰符： 目前固定 public static

​ 返回值类型： 目前固定， 用void。

​ 方法名： 就是我们自己给方法起的名字。 要满足标识符的规则以及规范。

​ 参数列表： 可以省略。 目前省略。

​ 方法体： 就是完成特定功能的代码块。

​ return 返回值： 方法调用完成之后的结果。 如果返回值类型是void，那么不能有返回值。所以说这一行可以省略。

​ 如果返回值类型是void，那么return 返回值 一般省略。

​ 如果不想省略， 那么只能单独写一个return， 不能再写后面的返回值。因为void表示没有返回值。

​ 注意：方法只定义是不会执行的，必须调用才会执行。

​ 方法的调用格式，对于返回值类型为void的方法，要使用以下格式调用：

​ 方法名(实际参数列表);

​ 如果在定义方法的时候，方法参数列表位置没有写东西，那么调用的时候小括号里面也什么都不写。

例如：

public class Demo01Method {
    public static void main(String[] args) {
        System.out.println("main...start");
        //调用用printMethod方法，调用这个方法，这个方法才会执行
        printMethod();

        System.out.println("main...end");
    }

    public static void printMethod() {
        System.out.println("你好");
        System.out.println("你今天早上洗头发了吗");
        System.out.println("哈哈哈");
    }
}

方法的注意事项。

\1. 方法和方法是平级关系，不能把一个方法定义在另一个方法里面

\2. 不能把一个方法定义在类的外面

3.+= 的特殊作用。

​ m+=n：是先计算m和n相加的结果。 然后把结果赋值给左边的变量m。 运算结果相当于 m = m + n;

​ 扩展的赋值运算符，比如+=， -=…..里面蕴含了强制类型转换的功能；

public class Demo01Ext {
    public static void main(String[] args) {
        int i = 3;
        int j = 2;
        //i += j;
        i = i + j;
        System.out.println("i:" + i); // 5
        System.out.println("j:" + j); // 2

        System.out.println("======================================");

        short s1 = 3;
        short s2 = 4;
        s1 += s2;
        //s1 = s1 + s2;
        System.out.println("s1:" + s1);

    }
}

常量优化机制， java编译器在编译时期， 对于常量的值是知道的。 所以编译时期会直接求出来常量运算的结果

public class Demo02Ext {
    public static void main(String[] args) {
        byte b1 = 3;
        byte b2 = 4;

        //byte b3 = 3 + 4;
        byte b3 = 7; //java中整数类型在赋值的时候， 如果在这个类型范围内，是可以直接赋值的。

        byte b4 = b1 + b2;// byte类型的变量相互运算会把自己提升为int，运算结果是int类型，不能赋值给byte
        System.out.println("b3:" + b3);
        System.out.println("b4:" + b4);

    }
}

Java中的类型转换。

Java中的类型转换有两种，一种是自动类型转换，另一种是强制类型转换。

​ 自动类型转换：

​ 取值范围小的类型在需要的时候可以自动转成取值范围大的类型。

​ byte,short,char -> int -> long -> float -> double

​ 注意：

​ \1. long虽然是8个字节，float是4个字节，但是long的取值范围没有float大。

​ \2. byte,short,char 他们之间不做转换。 只要这三个数据类型参与运算，都会先把自己提升为int类型。

​ 比如两个byte类型的变量相加也会自动把自己提升为int然后再去运算。

public class Demo01TypeCase {
    public static void main(String[] args) {
        int a = 3;
        int b = 4;
        System.out.println(a + b); //7

        //用一个变量接收a和b相加的结果，然后再打印
        int c = a + b;
        System.out.println(c); //7

        //定义两个不同类型的变量
        int aa = 3;
        byte bb = 4;
        System.out.println(aa + bb); //7

        //用一个变量接收aa和bb相加之后的结果
        //byte cc = aa + bb;  不可以
        int cc = aa + bb; //在java中，不同数据类型之间是不能直接运算的。 现在byte和int相加， 会把取值范围小的类型临时自动提升为取值范围大的类型，然后再去运算
        System.out.println(cc);
    }
}

Java中的强制类型转换

​ 强制类型转换：取值范围大的可以强转成取值范围小的。

​ 强制类型转换系统不会帮我们去做，我们需要使用格式去实现。

​ 格式：

​ 目标数据类型 变量名 = (目标数据类型)要强转的数据;

​ 强制类型转换是不推荐， 因为会损失精度

public class Demo02TypeCase {
    public static void main(String[] args) {
        //定义两个不同数据类型的变量
        int a = 300;
        byte b = 4;
        //byte c = a + b;
        byte c = (byte)(a + b); // 304  1 0011 0000    ->   0011 0000

        System.out.println(c);
    }
}

运算符就是用来对数据进行计算的符号

​ 算数运算符： + - * / % ++ –

​ +： 做加法

​ -： 做减法

​ *： 做乘法

​ /： 做除法。 如果两个整数相除，结果只取整数部分.如果想要得到小数部分， 只需要把除法两边任意一个改成小数即可

public class Demo01Operator {
    public static void main(String[] args) {
        //定义两个变量
        int a = 5;
        int b = 3;
        //做加法
        System.out.println(a + b);// 8
        //做减法
        System.out.println(a - b);// 2
        //做乘法
        System.out.println(a * b);//15
        //做除法
        System.out.println(a / b);// 1
        //带有小数部分的结果
        System.out.println(5 / 3.0);

    }
}
